# IoannisV - OPS

A modern, high-performance portfolio and professional website for **IoannisV - OPS**, showcasing expertise in cybersecurity, physical security, maritime operations, IoT engineering, software development, and security consulting.

Built with **Next.js 16**, **React 19**, **TypeScript**, and **Tailwind CSS 4**, the application delivers a fast, responsive, and accessible user experience while maintaining a clean and scalable architecture.

---

## Features

- Modern responsive design
- Next.js App Router architecture
- TypeScript throughout the project
- Tailwind CSS v4 styling
- Dark / Light theme support
- Professional portfolio sections
- Experience timeline
- Certifications showcase
- Project portfolio
- Blog support using Markdown
- RSS feed generation
- Contact integration with EmailJS
- SEO friendly structure
- Privacy policy page
- Smooth animations using Framer Motion
- Modular reusable UI components

## Installation

Clone the repository

```bash
git clone https://github.com/ioannisv-ops/main.git
```

Navigate into the project

```bash
cd main
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will be available at

```
http://localhost:3000
```

---

## Build

Production build

```bash
npm run build
```

Run production

```bash
npm start
```

Lint

```bash
npm run lint
```

---

## Environment Variables

Create a `.env` file from the provided example.

```
cp .env.example .env
```

Configure the required EmailJS credentials and any additional environment variables before deployment.

---

## Blog

Blog articles are stored inside

```
content/blog
```

Posts are written in Markdown and are automatically processed during build.

An RSS feed is generated during the build process.
